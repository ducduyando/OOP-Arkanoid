## **Description**

- The game is coded with Java and LibGdx framework
- You will control the paddle and the ball just like the ordinary Arkanoid, but the difference is we make this game become an RPG! Instead of boring bricks, there are 3 exciting bosses with special skills waiting for you and your mission is to defeat all them all!
- But ain't like the normal Arkanoid, this game is not easy to beat so to help the you, everytime you defeat a boss, you will have the chance to steal one of its abilities in order to help you with the next one
- This game focuses on countering and understand how bosses' abilities work, skillfully control the paddle to catch the ball while dodging the bosses' skill
- This upgraded version of Arkanoid is remade with different concepts, pixel art styles, 8bit and retro arcade soundtracks and sfxs

## **Binding buttons**

- You will control the paddle with WASD
- The abilities are controlled with J and K
